# 公众号封面图 Draw.io 生成指南

供 wechat-article-writer 生成封面/结尾图时参考。用户要求「生成封面」「封面图」或排版建议中包含【封面图】时使用。

## 使用时机

- 用户需要公众号文章封面图、结尾总结图
- 用户需要技术博客配图、B 站封面、小红书图文
- 排版建议中标注【封面图】【结尾图】需求

## Step 1：收集输入

从用户消息或文章内容提取：用途（封面图/结尾图）、比例（默认合并封面）、主题/主标题/副标题、配色偏好。

## 公众号封面尺寸规范

| 用途 | 比例 | 画布尺寸 |
|------|------|---------|
| 大封面（文章列表展示） | 2.35:1 | 900×383 px |
| 小封面（订阅号消息） | 1:1 | 383×383 px |
| **合并封面（推荐）** | — | **1283×383 px**（左侧 900 大封面 + 右侧 383 小封面） |

> **默认生成合并封面**：1283×383 px，左侧大封面区域（0–900）放主视觉，右侧小封面区域（900–1283）放方形构图。用户可按需裁剪导出两张。

## Step 2：生成 Draw.io XML

- **画布尺寸**：
  - 合并封面（默认）→ **1283×383**
  - 仅大封面 2.35:1 → **900×383**
  - 仅小封面 1:1 → **383×383**
  - 16:9（B 站/PPT）→ 1920×1080
  - 9:16（竖版/小红书）→ 1080×1920
- **合并封面布局**：
  - 全画布背景矩形（1283×383）
  - 左区（x:0 w:900）：主标题大字（56–72pt）+ 副标题（24–32pt）+ 装饰元素
  - 右区（x:900 w:383）：方形构图，主标题缩小版（36–48pt）+ 核心视觉，无副标题
  - 两区颜色一致，右区可加深背景或加边框以区分裁剪区域
- **装饰元素边界约束（严格执行）**：
  - 所有装饰元素（圆形、图标、色块等）的 x、y、width、height 必须使 **x ≥ 0、y ≥ 0、x+width ≤ 画布宽、y+height ≤ 画布高**
  - 禁止任何元素超出画布边缘，包括用负坐标或过大尺寸制造"溢出"效果
  - 圆形装饰半径不超过画布高度的 60%（合并封面中 ≤ 230px）
- **配色**：科技/教程/暖色/商务/清新/极简/活力/品牌色，用户未指定时根据主题推断
- **XML**：标签正确闭合、vertex="1"、id 从 0 递增、特殊字符转义

## Step 3：输出

1. 完整 mxGraph XML，可保存为 `.drawio` 文件
2. 封面说明：1–2 句布局用途与风格
3. 使用说明：保存为 .drawio → Draw.io 打开 → 编辑 → 导出 PNG/JPG

## 推荐风格：明亮活力风（默认首选）

参考样式：亮色渐变背景 + 大字混排 + 笔刷装饰，整体轻快明亮，避免深色背景。

### 配色方案

| 配色 | 背景渐变 | 强调色 |
|------|---------|-------|
| 清新青绿（默认） | `#B8F0D8` → `#A8E6CF` → `#C8F0C0`（左青绿到右薄荷） | `#FF8C00`（橙）|
| 活力黄橙 | `#FFF0A0` → `#FFD060` → `#FFB030` | `#E63C00`（深橙红）|
| 粉紫梦幻 | `#F8D0F0` → `#E8B8F8` → `#C8A8FF` | `#8800CC`（紫）|
| 天蓝清爽 | `#B8E8FF` → `#90D0FF` → `#C0F0F8` | `#0060CC`（蓝）|

### 布局要素

1. **背景**：双色或三色斜向渐变矩形，填满画布；可叠加一个半透明大圆（直径 220–300px，opacity 60–80%，颜色取渐变亮色）
2. **笔刷底纹**：2–3 条倾斜矩形（旋转 -5°~-8°，白色 fillColor，opacity 70–85%，height 60–90px），叠在标题文字下方作底板
3. **主标题**：英文用粗黑体（fontStyle=1，fontSize 64–80px，黑色），中文用大字（fontStyle=1，fontSize 52–68px，强调色）；两行错落叠放在笔刷上
4. **装饰**：2–4 个四角星（★ 或 ✦，fontSize 18–28px，黑色或强调色），散布在标题周围；右上角可放小圆形 logo 区（直径 60px，白色边框）
5. **署名**：画布底部居中，fontSize 22–26px，深灰色（`#444444`）

### 禁止事项

- **禁止深色背景**（`#000000`、`#1a1a2e`、`#0d0d0d` 等深色系），除非用户明确要求
- 装饰圆不超过 3 个，且全部在画布内（边界约束见上方规则）

## 注意事项

- 主标题需足够大，确保公众号小图预览可辨识
- 配色与正文插图保持一致，便于整体风格统一
- 用户未指定风格时，默认使用「清新青绿」明亮活力风
